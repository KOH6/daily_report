### Material UI

- hover 時にボタン名や色を変更する方法

```jsx
<Button
  sx={{
    width: "10rem",
    borderRadius: 50,
    // ホバー時にボタンデザインを変更する
    "& .hoverText": {
      display: "none",
    },
    "&:hover": {
      background: "#FFEDEC",
      borderColor: "#FEC9CE",
      transition: "0s",
      "& .defaultText": { display: "none" },
      "& .hoverText": { display: "inline" },
    },
  }}
>
  <p className="defaultText">フォロー中</p>
  <p className="hoverText" style={{ color: "#F4202E" }}>
    フォロー解除
  </p>
</Button>
```
